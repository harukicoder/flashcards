One-time setup for cloud sync

To sync across devices with Google Sign-In, create a free Firebase project (~3 min):

1Go to console.firebase.google.com → Add project
2Authentication → Get started → enable Google provider
3Firestore Database → Create database → Start in test mode
4Project Settings (gear icon) → Your apps → Web (</>) → register an app → copy the firebaseConfig object

Config is stored locally in this browser only.
